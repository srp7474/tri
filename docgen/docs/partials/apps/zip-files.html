To download these files use the <Right-Click>Save link as ...action on the link.

Note: The development environment in which these were used and configured was running Java 6. This is the reason for the use of the GAE SDK 1.8.1.1, the last version which supported Java 6.

Plans are made to upgrade to Java 7 at some point. Since the use by Trianglar of the facilities of the Google App Engine is bare bones this TODO item has not made it to the top of the list.

If you opt for the Node.js Variant you do not need Java or the GAE SDK 1.8.1.1 to be installed on your machine. You will need Node.js installed and in the default path on your development machine.

Triangular Complete Development Package Triangular Complete Development Package. The detailed installation instructions are at Tutorial. It will also require the **GAE SDK** which can be obtained with the links below.

To run the development package you:
  1. Start command line processor and set CWD to devp-unzip-drive:folder\bats
  2. Run run-ws GAE Variant
  3. Run run-tri Node.js Variant
This package contains both the Node.JS & GAE Variants

To use the GAE Variant you must have Java 6 or more runtime installed on your development machine.

To use the Node.JS Variant you must have Node.js installed and in the default path on your development machine.

Node.js can be downloaded from Node.js Install

Included in this package are waks.js and tri-server.js, the Javascript implementations for the WAKS Utility and the Node.js Tri-WebServer
