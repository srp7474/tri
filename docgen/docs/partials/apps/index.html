Introduction

For a typical CRUD application written in AngularJS, Triangular can boost the coding productivity by factoring out the common elements of such an application into a reusable infrastructure.

This demonstration site allows you to quickly get a feel for the capabilities of Triangular.

  • For application demonstrations, the guest userid is prefilled. Just press login
  • Applications are sandboxed. Changes you make will only reflect in your local cache.
  • The application data is computer generated. With the exception of the Hello application, it does not contain real names, addresses or phone numbers for people or businesses.
