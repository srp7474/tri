Overview

The Node.JS Variant comes with a Node.JS server.

This is an extension of the commonly used webserver.js code found on the web. It is found in the download zip Download NodeJS Variant as tri-server.js The extensions include:

  • Handling of more extension types

  • Startup params are more extensive

  • Handling of AJAX calls according to the Server Command Set

  • Handling of the data storage as defined in Datastore NodeJS
