Overview

Triangular is presently implemented under 2 server structures.

The first, the Google App Engine is referred to as the GAE Variant. Refer also to Datastore GAE and Google App Engine Implementation

The second, the Node.js implementation and hosted at NodeJitsu Node.js Hosting is referred to as the Node.js Variant. Refer also to Datastore NodeJS and Node.js Implementation and Node.js Home Page

The GAE Variant or Node.js Variant nomenclature is used in the documentation to highlight a section of text that applies only to a particular variant.
