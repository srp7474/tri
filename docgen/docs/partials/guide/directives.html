Overview

Directives are the AngularJS facility that allow for the easy extension of the display capabilities and behavoir of HTML pages.

Triangular uses this facility in several places. The templates along with the macro facility is the most complex use of directives. The blurEvent, focusClear and autotab directives are more simple examples.

Nothing in the design of Triangular prohibits the addition of custom directives to an application. The oApp.directive('directive name', function(...) { format should be used since oApp is a global variable in which all directives are registered.
