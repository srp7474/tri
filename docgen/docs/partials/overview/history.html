This page details the major releases of Triangular.

Initial Release - Nov/1/2013

The initial Google App Engine version was released. This is hosted at Google App Engine Host

Node.js Variant Released - Nov/15/2013

This release included a Node.JS backend server and allowed for the release in a pure Node.js environment with no requirement for Java on the development machine and host machine. This is hosted at NodeJitsu Host

HelloUI Added - Nov/30/2013

This release added the HelloUI App that demonstrates the integration of Triangular and the JQueryUI Interface

TRI.js passed JSHint validate - Dec/2/2013

It was realized that TRI.js needed to pass the JSHint validation in order to be acceptable to the Open Source Community.
