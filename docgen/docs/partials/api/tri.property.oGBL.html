oGBL (property in module tri )

Description

oGBL represents the single global variable used to make available global data and functions to all component types of the Triangular system.

The standard values stored in oGBL include:

msgHandlerThe Event Message Handler handlers
optTablesThe optional role based tables loaded in a selective fashion. See 'Admin.js'
templatesThe templates loaded by hookMsgListener and used by the Macro directive.

Usage

tri.property:oGBL
