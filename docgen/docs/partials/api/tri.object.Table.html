Table (type in module tri )

Description

Provides an object rendition of the Static Table object defined in the Javascript file.

Referred to in this documentation as the dynamic instance of the Table object.

The public methods attached to the Table object and described herein are accessible form the HTML code assuming a Table object is available.

Methods

  • Table(props, $scope, servCRUD)

    Constructor of the the Table object. Performs the following functions:

    • The oTabProps are copied to props
    • Filters are migrated and a Limit 50 is added if Limit is not specfied
    • Dynamic Col objects are also created.

    Parameters

    • props – {Object} –

      Properties array. Equivalent to oTabProps in static object.

    • $scope – {Object} –

      under which table is connected

    • servCRUD – {Object} –

      object pointer. Needed to prevent circular references.

  • findCol(sCol)

    Returns a Col whose col matches sCol

    Parameters

    • sCol – {Object} –

      The column name to match

    Returns

    {Object} –

    Col object

  • findCol(sType)

    Returns a Col array whose col type sType

    Parameters

    • sType – {Object} –

      The column type to match

    Returns

    {Object[]} –

    Col object array

  • getEditActions($scope, oObj)

    Returns an array of custom actions pertaining to the Edit operation on a data object in the table.

    Parameters

    • $scope – {Object} –

      Scope under which call is made. The pointer to the click function and whatClass function will be added to the scope.

    • oObj – {Object} –

      The object beding edited.

    Returns

    {Object[]} –

    array of Action objects

  • getFilters(sType)

    Builds an array of Filters that match sType

    Parameters

    • sType – {string} –

      The filter type to match. Null means all filters.

    Returns

    {Object[]} –

    array of Filters objects

  • getObjs()

    Returns an array of all the data objects for the Table.

    Returns

    {Object[]} –

    array of data objects

  • getOptActions($scope)

    Returns an array of optional actions specified in the props object.

    Parameters

    • $scope – {Object} –

      Scope under which call is made. The pointer to the click function and whatClass function will be added to the scope.

    Returns

    {Object[]} –

    array of Action objects

  • getSelObjs()

    Reduces the entire sorted set by the filter logic. The results are cached because this function is called many times during a $digest operation. Changes to the Filters or data contents of the Table automatically cleanse the cache.

    Returns

    {Object[]} –

    array of data objects

    Returns the recTitle value
